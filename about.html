About

This is a professional, multi-page static website built with plain HTML, CSS, and JavaScript. It’s designed to be easy to deploy to a git repository and to run on any static host.

Tech

  • Semantic HTML5
  • CSS with custom properties (variables) and a simple reset
  • Vanilla JavaScript (ES modules) for navigation and utilities
  • No build step — open the files or serve with any static server

Structure

Pages share a single header and footer. Styles are split into variables, base, and main CSS. Scripts are modular: main entry, navigation (including mobile menu), and shared helpers.
